PSP-Nx Dual

This block communicates with a PS2 controller using PSP-Nx interface sensor. It returns the four joystick positions, as well as the state of the 16 buttons on the PS2 controller (pressed or unpressed).

Inputs
Input Port- This is where you input the Input Port your PSP-Nx is attached to.

Outputs
All the outputs are in the cluster Joystick and Button Values
Left X & Y abd Right X & Y are the values of the joysticks.
The booleans are the individual button values if the button is pressed it will be true if not it will be false.
