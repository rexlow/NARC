NXTHID Remote

 
NXTHID is a Human Interface Device, initiating commands from NXT.


Inputs
Input Port -This is where you input the Input Port your NXTHID is attached to.
Transmit Mode- Specify if you want to use direct keyboard commands or have the HID type a written text.
If in Text Mode input a string in Text Data.
If in Direct Keyboard use Key Modifier annd Keydata.
