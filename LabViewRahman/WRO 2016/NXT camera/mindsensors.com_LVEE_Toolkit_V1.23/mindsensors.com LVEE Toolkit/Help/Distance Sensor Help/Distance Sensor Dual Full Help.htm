Distance Sensor Dual

The distance ranges of sensor variants are as follows:
  • Long Range: 20 to 150 cm
  • Medium Range: 10 to 80 cm
  • Short Range: 4 to 30 cm
The sensor detects distance based on angle of arriving reflected IR light.  The sensor emits a IR light, and when it reflects from the obstacle, it measures angle at which the light strikes the sensor.
This angle is used to determine the distance to the obstacle.
The sensor will not be able to detect an object that does not reflect IR Light (such as clear glass).


Inputs
Input Port - This is where you input the Input Port your Distance Sensor is attached to.

Outputs
Output of Acceleration Sensor (Cluster) includes...
Distance Value –This gives you the Distance in mm.
